üßæ Receipt Generation

Feature Description: When the order is finished, the system generates a receipt showing every item the customer selected, with a total cost.

Purpose of Code: This class handles writing order details to a text file in the receipts/ folder. The filename includes the timestamp for uniqueness.

public class ReceiptService {
    public void saveReceipt(Order order) {
        String fileName = LocalDateTime.now().format(DateTimeFormatter.ofPattern("yyyyMMdd-HHmmss")) + ".txt";
        try (BufferedWriter writer = new BufferedWriter(new FileWriter("receipts/" + fileName))) {
            writer.write(order.toString());
        } catch (IOException e) {
            System.out.println("Error saving receipt: " + e.getMessage());
        }
    }
}

Explanation: This class creates a new receipt file using the current date and time, then saves the customer's order details into that file.
